# 附录 E：数学基础

现代深度强化学习的强大之处在于，很多工作可以不依赖深层数学理解就能完成。调一个 PPO 训练脚本、跑通一个 RLHF pipeline、在 SWE-bench 上评测——这些操作层面的工作，数学门槛并不高。

但当你需要理解"为什么 PPO 的裁剪系数设成 0.2""为什么策略梯度的方差这么大""为什么 KL 散度能防止奖励黑客"时，公式背后的数学就变得不可回避了。这不是为了考试，而是为了在面对训练不收敛、策略退化、奖励崩溃等问题时，能从原理出发诊断和修复。

## 本附录覆盖什么

附录按数学领域分为四个部分，每部分聚焦"RL 最需要的那些数学"：

| 部分 | 主题 | 你会在哪里用到 |
| --- | --- | --- |
| [E.1 线性代数](./linear-algebra) | 向量、矩阵、点积、特征值、范数 | 状态表示、神经网络参数、价值函数收敛分析 |
| [E.2 概率与统计](./probability-statistics) | 条件概率、贝叶斯、期望、方差、常用分布、蒙特卡洛 | MDP 形式化、策略梯度、GAE 的偏差-方差权衡 |
| [E.3 微积分与优化](./calculus-optimization) | 导数、链式法则、梯度下降、Adam、凸优化 | 策略梯度定理、PPO/GRPO 的梯度分析 |
| [E.4 信息论](./information-theory) | 熵、KL 散度、交叉熵 | SAC 熵正则化、RLHF 的 KL 约束、DPO 隐式奖励 |

## 怎么用这个附录

这个附录不是教材。它不要求你从头读到尾。推荐的用法是：

1. **随用随查**。读正文遇到不熟悉的数学符号时，回到对应章节。
2. **按需深入**。每个概念从直觉出发，到形式化定义，再到 RL 中的具体应用。你可以停在任何一个层级——只看直觉就够用，也可以一路读到推导细节。
3. **公式速查**。每个子页面末尾附有该领域的核心公式汇总表，方便快速查找。

如果你是第一次接触这些数学，建议从 [E.2 概率与统计](./probability-statistics) 开始——它是理解全书大部分内容的基础。
