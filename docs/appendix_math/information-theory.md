# E.4 信息论

> 相关章节：[第6章 GAE 与奖励模型](/chapter06_ppo/gae-reward-model)、[第7章 DPO](/chapter07_alignment/dpo-math)、[第9章 SAC](/chapter09_continuous_control/sac-comparison)

信息论是关于"信息量如何度量"的数学。它为 RL 提供了三个关键工具：熵（度量不确定性）、KL 散度（度量两个分布的差距）、交叉熵（分类任务的标准损失）。SAC 用熵鼓励探索，RLHF 用 KL 约束防止策略跑偏，DPO 用隐式的 KL 正则化绕过奖励模型。

## 自信息：信息 = 惊讶程度

考虑一个思想实验。你的朋友从一副洗好的牌中翻牌：

- "我看到了一张牌。" —— 你一点也不惊讶，这件事概率为 1，没有提供任何**新信息**。
- "这是一张红心。" —— 你略微惊讶。四种花色等概率，这件事的概率是 $1/4$。
- "这是黑桃3。" —— 更惊讶了。52 张牌等概率，概率是 $1/52$。
- 朋友按顺序读出整副牌的花色和点数。你极度惊讶。$52!$ 种排列等概率，概率小到难以想象。

规律：**事件越不可能发生，当你得知它发生时，获得的信息就越多。** 信息量与概率的倒数成正比。

Shannon 在 1948 年给出了数学化的定义——自信息（self-information）：

$$I(X) = -\log_2 P(X)$$

用对数是因为信息的可加性：两个独立事件同时发生的信息量，应该等于各自信息量之和。$\log(P(A) \cdot P(B)) = \log P(A) + \log P(B)$。负号保证信息量为正（概率在 $(0, 1]$ 之间，对数为负）。

单位是 **bit**（以 2 为底时）。回到扑克牌的例子："看到一张牌"信息量为 $-\log_2(1) = 0$ bit；"看到红心"是 $-\log_2(1/4) = 2$ bit；"看到黑桃3"是 $-\log_2(1/52) \approx 5.7$ bit。

## 熵：平均不确定性

自信息度量的是**单个事件**的信息量。熵（Entropy）度量的是**整个分布**的平均信息量——也就是这个随机变量的"平均惊讶程度"：

$$H(X) = -\sum_x P(x) \log P(x) = \mathbb{E}_{x \sim P}[-\log P(x)]$$

对于连续分布，求和变成积分：

$$H(X) = -\int p(x) \log p(x) \, dx$$

这叫微分熵（differential entropy）。

**直觉**：

- 熵高 → 分布均匀 → 每个结果都差不多让你惊讶 → 不确定性大
- 熵低 → 分布集中 → 大多数时候结果可预测 → 不确定性小

一个确定性分布（$P(x_0) = 1$，其他都为 0）的熵为 0——没有任何不确定性。均匀分布（所有结果等概率）的熵最大——不确定性最大。

### 熵与探索

在 RL 中，策略 $\pi(a \mid s)$ 的熵是：

$$H(\pi(\cdot \mid s)) = -\sum_a \pi(a \mid s) \log \pi(a \mid s)$$

- 熵高：策略在各个动作之间犹豫不决，探索充分
- 熵低：策略几乎确定性地选一个动作，利用充分但可能陷入局部最优

**SAC（Soft Actor-Critic）** 把熵直接加到目标函数中：

$$J(\pi) = \sum_{t=0}^{T} \mathbb{E}\left[r(s_t, a_t) + \alpha \cdot H(\pi(\cdot \mid s_t))\right]$$

系数 $\alpha$ 控制"探索 vs 利用的权衡"。$\alpha$ 大 → 更鼓励探索；$\alpha$ 小 → 更关注即时回报。SAC 甚至可以自动调整 $\alpha$，通过维持目标熵水平来实现。

PPO 和 GRPO 的训练中也常见**熵正则化**——在损失函数中加一个 $-\beta H(\pi)$ 项，防止策略过早收敛到确定性策略。

## 交叉熵：用一个分布编码另一个分布

交叉熵度量的是：用分布 $Q$ 去编码来自分布 $P$ 的数据时，平均需要多少信息量：

$$H(P, Q) = -\sum_x P(x) \log Q(x)$$

直觉：$P$ 是真实分布，$Q$ 是你的模型。如果 $Q$ 完美地匹配 $P$（$Q = P$），交叉熵就等于熵 $H(P)$。如果 $Q$ 偏离 $P$，交叉熵会大于熵——你需要更多的信息来弥补模型的偏差。

**为什么交叉熵是分类任务的标准损失？** 分类任务中，$P$ 是 one-hot 真实标签，$Q$ 是模型输出的概率分布。最小化交叉熵 $H(P, Q)$ 等价于最大化正确类别的对数似然 $\log Q(y_{true})$。直觉上就是"让模型对正确答案给出更高的概率"。

在 RL 中，策略蒸馏（policy distillation）用交叉熵让 student 策略模仿 teacher 策略：最小化 $H(\pi_{teacher}, \pi_{student})$。

## KL 散度：两个分布之间的"距离"

KL 散度（Kullback-Leibler Divergence）度量的是：从 $P$ 的角度看，$Q$ 和 $P$ 差多少：

$$D_{KL}(P \| Q) = \sum_x P(x) \log \frac{P(x)}{Q(x)} = H(P, Q) - H(P)$$

三个关键性质：

1. **非负性**：$D_{KL}(P \| Q) \ge 0$，当且仅当 $P = Q$ 时取等号
2. **不对称性**：$D_{KL}(P \| Q) \ne D_{KL}(Q \| P)$。这不是一个真正的距离度量
3. **交叉熵 = 熵 + KL 散度**：$H(P, Q) = H(P) + D_{KL}(P \| Q)$

### 不对称性的直觉

$D_{KL}(P \| Q)$ 和 $D_{KL}(Q \| P)$ 关注的东西不同：

- $D_{KL}(P \| Q)$：在 $P$ 概率大的地方，$Q$ 必须也给出合理的概率。如果 $P$ 说某事件概率很高但 $Q$ 认为几乎不可能，$D_{KL}(P \| Q)$ 会很大。这种模式下 $Q$ 必须覆盖 $P$ 的所有"重要区域"。
- $D_{KL}(Q \| P)$：在 $Q$ 概率大的地方，$P$ 不能说几乎不可能。如果 $Q$ 在 $P$ 认为不可能的区域给出概率，$D_{KL}(Q \| P)$ 会趋向无穷。这种模式下 $Q$ 被"零均值约束"所限制，不能偏离 $P$ 太远。

### KL 散度在 RLHF 中的核心角色

RLHF 训练中，奖励模型可能被"钻空子"——策略可以生成看似高分但实际无意义的回答。KL 散度约束是防御手段：

$$J(\pi) = \mathbb{E}_{\pi}[r(x, y)] - \beta D_{KL}(\pi_\theta \| \pi_{ref})$$

$\beta D_{KL}$ 项惩罚策略偏离参考模型 $\pi_{ref}$（通常是 SFT 后的模型）太远。这确保模型在"追求高分"和"保持正常语言能力"之间取得平衡。

### DPO 的隐式 KL 正则化

DPO 看起来没有 KL 散度项，但它通过 Bradley-Terry 模型推导出的损失函数天然包含了 KL 正则化：

$$\mathcal{L}_{DPO} = -\mathbb{E}\left[\log \sigma\left(\beta \log \frac{\pi_\theta(y_w \mid x)}{\pi_{ref}(y_w \mid x)} - \beta \log \frac{\pi_\theta(y_l \mid x)}{\pi_{ref}(y_l \mid x)}\right)\right]$$

$\beta \log (\pi_\theta / \pi_{ref})$ 这一项就是隐式奖励，而 $\beta$ 控制的是等价于 KL 惩罚系数的效果。$\beta$ 越大，策略越不能偏离参考模型。

## 互信息

互信息度量两个随机变量之间的"信息关联"：

$$I(X; Y) = D_{KL}(P_{XY} \| P_X \otimes P_Y) = H(X) - H(X \mid Y)$$

直觉：知道 $Y$ 之后，$X$ 的不确定性减少了多少。如果 $X$ 和 $Y$ 独立，$I(X; Y) = 0$；如果知道 $Y$ 就完全确定 $X$，$I(X; Y) = H(X)$。

在 RL 中，互信息用于表征学习（representation learning）——学习到的状态表征 $\phi(s)$ 应该与未来回报有高互信息 $I(\phi(s); G_t)$，同时与无关特征有低互信息。

## 公式速查

| 概念 | 公式 | RL 中的角色 |
| --- | --- | --- |
| 自信息 | $I(X) = -\log P(X)$ | 信息量的基本度量 |
| 熵 | $H(X) = -\sum P(x)\log P(x)$ | SAC 熵正则化、探索策略 |
| 交叉熵 | $H(P,Q) = -\sum P(x)\log Q(x)$ | 分类损失、策略蒸馏 |
| KL 散度 | $D_{KL}(P\|Q) = \sum P(x)\log\frac{P(x)}{Q(x)}$ | RLHF KL 约束、PPO/GRPO |
| 互信息 | $I(X;Y) = H(X) - H(X\|Y)$ | 表征学习 |
| 隐式奖励 | $r(x,y) = \beta\log\frac{\pi_\theta(y\|x)}{\pi_{ref}(y\|x)}$ | DPO 的隐式 KL 正则化 |
