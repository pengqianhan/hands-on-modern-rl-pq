# 第15章：未来趋势

> **本章目标**：展望 RL 领域正在发生的前沿变化，帮助你在学完本书后找到持续深入的方向。

## 15.1 测试时计算扩展：从训练 RL 到推理时搜索

- 传统 RL：训练时消耗算力，推理时直接输出
- 新范式：推理时也做搜索（如 Tree of Thought、MCTS）
- 代表性工作：OpenAI o1/o3 的"思考链"——推理时用 RL 训练的策略进行多步搜索
- 意义：算力可以从"训练更多"转向"推理时想更久"
- **你将思考**：RL 的价值不只在训练，还在推理时的决策

## 15.2 多模态与具身智能的 RL 融合

- **机器人 RL**：
  - 从仿真到现实（Sim-to-Real）：在模拟器中训练，迁移到真实机器人
  - 代表工作：Google 的 RT-2（用 VLM 直接控制机器人）
- **自动驾驶中的 RL**：
  - 端到端驾驶策略的 RL 训练
  - 安全约束下的策略优化（不能像游戏一样随便试错）
- **具身智能的挑战**：
  - 真实世界无法快速重置（不能像游戏一样 reset）
  - 安全约束：某些动作永远不能尝试
  - 样本效率：真实世界交互成本极高
- **你将思考**：从数字世界到物理世界，RL 面临的根本性变化

## 15.3 RL 与推理能力的关系

- DeepSeek-R1 的启示：RL 训练可以涌现出"自我验证"、"反思修正"等推理能力
- 这些能力是从哪里来的？——不是人为设计的，而是 RL 优化过程中自然涌现的
- 开放问题：推理能力的上限在哪里？是否可以通过更多 RL 训练持续提升？
- **你将思考**：RL 不仅是"对齐工具"，还可能是"推理能力的催化剂"

## 15.4 开源社区资源与学习路径

- **入门推荐**：
  - 本书 + 配套代码仓库
  - Gymnasium（原 OpenAI Gym）官方文档
  - Stable-Baselines3（RL 算法库）的教程
- **进阶推荐**：
  - 深入阅读 PPO、DPO、GRPO 的原始论文
  - HuggingFace TRL 库（LLM 对齐工具）
  - Verl / OpenRLHF（工业级 RLHF 框架）
- **研究方向**：
  - 高效 RL 训练（减少采样量、降低显存）
  - 安全 RL（约束优化、红队测试）
  - 多智能体 RL（协作与竞争）
- **你将获得**：从本书出发的持续学习路线图

## 小结

- RL 正在从"训练时优化"扩展到"推理时搜索"，推理算力成为新的优化维度
- 多模态和具身智能为 RL 开辟了从数字世界到物理世界的应用场景
- RL 训练涌现的推理能力可能是通向更强 AI 的一条路径
- 开源社区提供了丰富的工具和学习资源，持续实践是最好的学习方式
