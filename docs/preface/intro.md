# 前言

2016 年 3 月，AlphaGo 以 4:1 击败李世石，强化学习第一次以震撼人心的方式进入公众视野。那一刻，全世界都在问同一个问题：一个程序是怎么"学会"下围棋的？然而在那之后的几年里，对绝大多数工程师而言，强化学习仍然停留在新闻标题里。想入门 RL，你面前通常只有两条路：一条是经典教材——学术上严谨，但通篇是形式化定义和数学推导，读完半本书可能还没写过一行训练代码；另一条是散落在 GitHub 上的各种 RL 实现仓库——代码跑得通，但缺少对"为什么这样设计"的解释。两条路之间的鸿沟，让大量有志于此的学习者望而却步。

2022 年底，ChatGPT 的发布改变了一切。人们忽然发现，强化学习不只是让 AI 下棋和玩游戏——它正是让大语言模型从"能说话"变成"说好话"的关键技术。从 InstructGPT 到 ChatGPT，从 DeepSeek-R1 到各类开源对齐模型，RLHF、DPO、GRPO 等算法已经深刻地重塑了整个 AI 行业的工作方式。与此同时，RL 在自动驾驶、机器人控制、推荐系统、科学发现等领域也取得了实质性进展。强化学习不再是学术实验室的专属领地，它已经成为每一位 AI 从业者都需要理解的核心工具。

然而，一个令人遗憾的现实是：**市面上的学习资源严重滞后于行业实践**。主流的深度学习教程几乎都将注意力集中在监督学习上，对 RL 一笔带过甚至完全不涉及。而专门的 RL 教材又往往停留在传统框架（动态规划、表格方法），对 PPO、DPO、GRPO 这些当前工业界每天都在使用的算法只字不提。一个想要理解 RLHF 流程的工程师，不得不在经典教材和最新的研究论文之间艰难地自行搭建桥梁。这本书就是为填补这道鸿沟而写的。

## 什么是强化学习？

如果你之前学过机器学习，你可能熟悉监督学习——给模型一堆标注好的数据（图片和标签、问题和标准答案），让它从中学会映射关系。强化学习解决的是另一类问题：**没有标准答案，只有"做得好不好"的反馈信号**。

举个最简单的例子。假设你在学骑自行车：
- **监督学习**的方式是：有人提前写好一本《骑车手册》，告诉你"左倾 3 度时向右施力 5 牛顿"，你照着做就行。
- **强化学习**的方式是：没人告诉你该怎么操作，你只能自己试——摔倒了就知道"刚才那个动作不好"，骑稳了就知道"刚才那个动作不错"。通过反复尝试和反馈，你逐渐学会了骑车。

这就是强化学习的核心循环：**智能体（Agent）在环境中采取动作（Action），环境返回奖励（Reward）和新状态（State），智能体根据这些反馈调整自己的策略（Policy），逐步学会在给定任务中做出更好的决策。**

这个框架惊人地通用。把"骑车"换成"下围棋"，智能体就是 AlphaGo；换成"玩超级玛丽"，智能体就是 DQN；换成"回答人类问题"，智能体就是 ChatGPT 背后的语言模型。从游戏到机器人，从推荐系统到大语言模型对齐，强化学习是"从试错中学习"这一思想的数学化表达。

与监督学习相比，强化学习有三个独特之处：

1. **没有标注数据**——没有人告诉你"在状态 s 下，最优动作是 a"，你只有"刚才那一连串动作最终得了多少分"的延迟反馈。
2. **序列决策**——你现在的动作会影响未来的状态，不是独立的单步预测。今天多吃一块蛋糕，影响的不仅是今天的体重，还有明天的运动计划。
3. **探索与利用的权衡**——你需要在"尝试新策略看看会不会更好"（探索）和"坚持已知的好策略"（利用）之间不断权衡。永远探索则浪费时间，永远利用则可能错过更优解。

这三个特点让强化学习比监督学习更难，但也更强大——它能让 AI 学会没有人教过它的行为。

## 关于本书

这本书代表了我们的尝试——**让现代强化学习变得平易近人，用代码、数学和直觉的融合来教会人们核心概念**。

### 一种"先动手、后理论"的学习路径

许多教科书按照逻辑严密性的顺序来组织内容：先讲完 MDP 的全部性质，再讲贝尔曼方程，再讲动态规划，最后才允许你碰一行代码。这种方式对研究者或许合适，但对工程师和初学者来说，漫长的前置等待往往消磨了学习的动力。

在这本书中，我们反其道而行。**你将从第一章的第一行代码开始训练一个智能体**。我们相信，当学习者亲眼看到 CartPole 的小车从摇摇晃晃到稳稳站立，亲手用 DPO 让一个大模型学会"说好话"，再回过头理解其背后的数学结构时，学习过程会更加自然，理解也会更加持久。

具体而言，每一章都遵循"动手先行 → 观察现象 → 理论学习 → 回看实践"的四步循环：

1. **动手先行**——给你一段可运行的代码或一个可操作的实验，让你先获得直接经验。
2. **观察与疑问**——引导你关注训练曲线上的关键现象，并提出"为什么会这样"的问题。
3. **理论学习**——在具备直觉的基础上，系统地讲解数学原理。
4. **回看实践**——用新学到的理论重新解读之前观察到的现象，完成从直觉到形式化的闭环。

### 代码与理论并重

本书的每一章都包含可运行的代码示例。我们之所以坚持这一点，是因为强化学习中的许多直觉只能通过试错来建立——调一调学习率，观察 reward 曲线的振荡；改一改 clip 参数，看看策略是否会崩溃。这些经验无法仅靠阅读公式来获得。

### 内容和结构

全书大致可分为四个部分：

**第一部分：快速入门（第1-2章）。** 第1章提供传统强化学习的入门体验——你将运行一个 CartPole 训练脚本，亲眼看到小车从摇摇晃晃到稳稳站立，在实验中理解状态、动作、奖励、策略等基本要素的含义。第2章则打开另一扇门：用 DPO 算法对 Qwen2.5-0.5B 进行偏好微调，直观体验大语言模型后训练（post-training）的完整流程，理解预训练、SFT、RL 三阶段各自的定位与局限。这两章不要求任何先验知识，目标是让你在十分钟内获得"RL 能做什么"的第一手感受。

**第二部分：理论基础（第3-4章）。** 在动手经验的基础上，我们系统地引入强化学习的数学框架。第3章通过一个简化的猜硬币游戏，将 MDP 的五元组（状态、动作、转移概率、奖励、折扣因子）逐一拆解，推导贝尔曼方程，并建立状态价值函数 V(s) 与动作价值函数 Q(s, a) 的直觉——前者衡量"当前局面值多少分"，后者衡量"当前局面做某个动作值多少分"。第4章从策略梯度的基本思想出发，推导 REINFORCE 算法，揭示其高方差问题的根源，进而引入基线（baseline）与优势函数 A(s, a)，最终构建 Actor-Critic 架构——这一架构将在后续章节中反复出现，成为 PPO、GRPO 等算法的共同骨架。有了第一部分的实践经验作为铺垫，这些理论的引入会显得水到渠成。

**第三部分：核心算法（第5-10章）。** 接下来的六章集中讨论当前工业界最重要的强化学习算法。第5章深入剖析 PPO（近端策略优化），它是当前使用最广泛的 on-policy 算法——从 OpenAI 的 RLHF 到各类游戏 AI 都依赖它。你将理解裁剪机制如何防止策略崩溃，GAE（广义优势估计）如何在偏差与方差之间取得平衡，以及 PPO 在 LLM 对齐中的对应关系。第6章转向 DPO（直接偏好优化），它巧妙地绕过了奖励模型训练这一工程瓶颈，将 RLHF 的问题直接转化为一个简单的分类损失。第7章介绍 GRPO（群体相对策略优化），这是 DeepSeek-R1 的核心创新——用组内相对比较替代 Critic 网络，大幅降低显存开销，使大规模推理增强训练成为可能。第8章是一个实用的算法选型指南，帮助你在不同场景下做出合理选择。第9章回到基于价值的方法，从经典 Q-Learning 出发，引入深度 Q 网络（DQN），解析经验回放与目标网络两大支柱机制，并延伸至 Double DQN、Dueling DQN、Rainbow 等改进谱系。第10章则进入连续动作空间的世界，在 MuJoCo 环境中对比 PPO、DDPG、TD3 等方法，理解高斯策略与确定性策略的差异，以及向量化环境带来的并行采样加速。每个算法都配有完整的代码实现、训练曲线分析和常见调试技巧。

**第四部分：前沿应用（第11-15章）。** 最后五章将强化学习基础与当前最热门的应用场景对接。第11章从工程视角全景展示 LLM 后训练的 RLHF 三阶段流水线——SFT（监督微调）、RM（奖励模型训练）、PPO（策略优化），并深入讨论奖励函数设计、奖励黑客的识别与防范、KL 散度惩罚等关键实践。第12章将 RL 的触角延伸到视觉-语言模型（VLM），用 GRPO 训练 VLM 回答视觉问题，讨论视觉 token 与文本 token 的奖励分配难题。第13章探讨 Agentic RL——训练智能体学会使用工具（如计算器、搜索引擎），理解动作空间扩展与多轮交互的 MDP 建模。第14章是一个实用的调试指南，逐一复现并解决策略崩溃、奖励黑客、显存溢出、训练不收敛四类最常见的训练故障。第15章展望未来趋势——从测试时计算扩展到多模态与具身智能的 RL 融合，帮助读者在学完本书后找到持续深入的方向。

### 目标读者

本书面向以下人群：

- **学生**（本科生或研究生）——希望系统掌握强化学习的理论基础和动手能力。
- **工程师**——希望理解 RLHF、DPO、GRPO 等算法的原理，并能在自己的项目中应用。
- **研究者**——希望快速了解现代 RL 的算法谱系，为深入研究奠定基础。

我们不要求读者具有强化学习或机器学习的背景。但假设读者了解以下基础知识：

- 基本的 Python 编程能力
- 线性代数（矩阵运算、向量内积）
- 微积分（偏导数、链式法则）
- 概率论基础（期望、条件概率、常见分布）

对于需要复习这些数学知识的读者，我们在附录 A 中提供了简要的速查手册。

## 小结

- 强化学习已经从实验室走向工业界，成为大语言模型后训练、机器人控制、游戏 AI 等领域的核心技术。
- 本书采用"先动手、后理论"的教学路径，每章都包含可运行的代码和系统化的数学讲解。
- 全书覆盖从 MDP、Q-Learning 到 PPO、DPO、GRPO，再到 LLM 对齐和智能体 RL 的完整知识图谱。
- 只需基本的 Python 编程和数学基础即可开始学习。
